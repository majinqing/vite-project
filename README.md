# Vue 3 + TypeScript + Vite

这个模板应该能帮助你开始使用 Vue 3 和 TypeScript 在 Vite 中进行开发。
该模板使用 Vue 3 的 `<script setup>` 单文件组件，
查看 [script setup 文档](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) 
了解更多信息。

在 [Vue 文档 TypeScript 指南](https://vuejs.org/guide/typescript/overview.html#project-setup) 
中了解更多关于推荐的项目设置和 IDE 支持。
